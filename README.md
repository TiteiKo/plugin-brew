brew
====

Plugin for [Oh My Fish](https://www.github.com/fish-shell/oh-my-fish) to autocomplet commands of [Homebrew](http://brew.sh).

Install
-------

```fish
$ omf install brew
```

Usage
-----

```fish
$ brew ins tap TAB
```
